Note over CSKH,AI: Đẩy case mới → KB

Mô hình AI Confidence

$$\text{confidence}(q, c) = \frac{|K_q \cap K_c|}{|K_q \cup K_c|}$$

Jaccard similarity — so khớp keywords câu hỏi với case trong KB

conf > 0.8
→ Tạo Mock PR tự động
→ CSKH chỉ cần review
conf ≤ 0.8
→ Escalate đội CSKH
→ Xử lý thủ công
$$\frac{\text{auto}}{N} \xrightarrow{\;\text{KB}\;} 1$$

Tỉ lệ tự động tăng dần

$$|KB_{t+1}| = |KB_t| + n$$
