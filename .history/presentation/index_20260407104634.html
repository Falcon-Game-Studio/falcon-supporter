Luồng Escalate

$\text{confidence} \leq 0.8$ → Chuyển đội CSKH xử lý thủ công

sequenceDiagram actor User participant FE as Chat UI participant BE as Backend participant AI as AI Agent participant KB as Knowledge Base participant CSKH as CSKH participant DB as JSON DB User->>FE: Gửi tin nhắn FE->>BE: POST /api/chat BE->>AI: analyze_message() AI->>KB: search_kb() KB-->>AI: no match AI-->>BE: {reply, status: escalated} BE-->>FE: Chuyển CSKH FE-->>User: Đang chuyển hỗ trợ... CSKH->>DB: Xử lý thủ công CSKH-->>FE: Phản hồi kết quả FE-->>User: Đã xử lý! CSKH->>AI: Đẩy case mới AI->>KB: Cập nhật KB

Mô hình AI Confidence

$$\text{confidence}(q, c) = \frac{|K_q \cap K_c|}{|K_q \cup K_c|}$$

$K_q$: keywords câu hỏi  |  $K_c$: keywords trong case KB  |  Jaccard similarity

$\text{conf} > 0.8$
→ Tạo Mock PR tự động
→ CSKH chỉ cần review Diff
$\text{conf} \leq 0.8$
→ Escalate đội CSKH
→ Xử lý thủ công
$$\frac{\text{auto-resolved}}{\text{total}} \xrightarrow{\text{KB grows}} 1 \qquad \Longrightarrow \qquad |KB_{t+1}| = |KB_t| + \Delta_{\text{new cases}}$$

Tỉ lệ auto-resolve tăng dần khi Knowledge Base phát triển
